# marp-theme

## 使い方

VSCodeの場合

[Marp for VSCode](https://marketplace.visualstudio.com/items?itemName=marp-team.marp-vscode)を使う場合、以下のようにテーマを設定できます。

```json
{
  "markdown.marp.themes": [
    "https://raw.githubusercontent.com/mofmof/marp-theme/main/mofmof.css"
  ]
}
```
